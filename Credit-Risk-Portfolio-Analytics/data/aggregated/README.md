# Aggregated Data

This directory contains only aggregated analytical outputs that are suitable for public portfolio presentation.

The repository does not include:

- Original Kaggle competition data
- Cleaned customer-level data
- Customer-level feature tables
- Customer-level predictions
- Kaggle customer identifiers
- Full modelling datasets

Planned aggregated outputs:

- Risk-level portfolio summary
- Financial risk summary
- Credit history summary
- LightGBM feature importance
